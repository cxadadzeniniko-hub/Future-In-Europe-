# Future In Europe
Future In Europe არის საგანმანათლებლო ვებგვერდი, რომელიც შექმნილია სტუდენტებისთვის ევროპაში სწავლის შესაძლებლობების გასაცნობად. ვებგვერდი მოიცავს ინფორმაციას უნივერსიტეტების, მიღების მოთხოვნების, სტიპენდიების და სტუდენტური ცხოვრების შესახებ ევროპის სხვადასხვა ქვეყანაში.
